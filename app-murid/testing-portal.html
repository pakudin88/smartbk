Quick Actions

Cara Menjalankan Aplikasi

  1. Buka Command Prompt atau PowerShell
  2. Navigasi ke folder aplikasi:
    cd c:\xampp\htdocs\simaklah-main\app-murid
  3. Jalankan salah satu perintah:
    php spark serve
    php spark serve --port=9000
    php spark serve --port=8082
  4. Atau jalankan file run-app-murid.bat dengan double-click

Troubleshooting

  • Pastikan menjalankan dari folder app-murid
  • Cek file Routes.php sudah benar
  • Coba clear cache: php spark cache:clear
  • Pastikan Auth controller dan view login ada

  • Periksa Routes.php configuration
  • Coba akses langsung: /login
  • Pastikan mod_rewrite aktif (jika menggunakan Apache)
  • Cek .htaccess di folder public/

  • Pastikan XAMPP MySQL running
  • Cek Database.php configuration
  • Pastikan database 'simaklah_db' sudah dibuat
  • Jalankan setup-murid-data.php untuk membuat tabel dan data
