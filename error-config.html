Configuration Error

File konfigurasi aplikasi tidak ditemukan atau tidak valid. Pastikan file .env sudah dikonfigurasi dengan benar.

Solusi:

1. Pastikan file .env ada di folder aplikasi:

app-guru/.env
app-ortu/.env
app-superadmin/.env

2. Pastikan konfigurasi baseURL sudah benar:

app.baseURL = 'http://localhost:8081/'

3. Contoh konfigurasi lengkap:

CI_ENVIRONMENT = development
app.baseURL = 'http://localhost:8081/'
database.default.hostname = srv1412.hstgr.io
database.default.database = u809035070_simaklah
Kembali ke Portal Refresh
Pastikan semua file konfigurasi sudah diatur dengan benar
