Test Sistem Redirect

Instruksi Testing
  1. Pastikan Apache sedang berjalan di XAMPP
  2. Jalankan server app-guru di port 8081: php spark serve --port=8081
  3. Jalankan server app-ortu di port 8080: php spark serve --port=8080
  4. Klik link test di bawah untuk memverifikasi redirect

Command Reference

Start app-guru server:
cd app-guru
php spark serve --port=8081
Start app-ortu server:
cd app-ortu
php spark serve --port=8080
Start all servers (batch file):
start-all-servers.bat
Check port usage:
netstat -an | findstr 808

Auto Status Check

Status akan diupdate otomatis setiap 5 detik. Pastikan server sudah berjalan untuk hasil yang akurat.
