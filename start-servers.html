Start SmartBK Servers

Sistem Folder-Based URLs: URL tetap sesuai struktur folder, tanpa redirect ke port!

Step 1: Start App-Guru Server

Buka Command Prompt atau PowerShell, jalankan:

cd c:\xampp\htdocs\smartbk\app-guru
php spark serve --port=8081

Tunggu hingga muncul: CodeIgniter development server started: http://localhost:8081

Step 2: Start App-Ortu Server

Buka Command Prompt atau PowerShell kedua, jalankan:

cd c:\xampp\htdocs\smartbk\app-ortu
php spark serve --port=8080

Tunggu hingga muncul: CodeIgniter development server started: http://localhost:8080

Step 3: Verify Server Status

Check apakah kedua server sudah berjalan:

App-Guru (Port 8081): Checking...

App-Ortu (Port 8080): Checking...

Troubleshooting

Problem: "Server not responding"

Pastikan server CodeIgniter sudah berjalan di terminal

Problem: "Port already in use"
taskkill /f /im php.exe
# Kemudian start server lagi
Problem: "Still showing redirect"

Check apakah .env file sudah benar:

  • app-guru/.env: app.baseURL = 'http://localhost:8081/'
  • app-ortu/.env: app.baseURL = 'http://localhost:8080/'
