Server Tidak Berjalan

Aplikasi server belum dijalankan atau tidak dapat dijangkau. Silakan start server terlebih dahulu untuk mengakses aplikasi.

Status: Server tidak merespons pada URL yang dikonfigurasi
Cara Menjalankan Server:

1. Buka terminal/command prompt

2. Masuk ke folder aplikasi dan jalankan server:

# Untuk Portal Guru:
cd app-guru
php spark serve --port=8081

# Untuk Portal Orang Tua:
cd app-ortu
php spark serve --port=8080

# Untuk Portal Super Admin:
cd app-superadmin
php spark serve --port=8082

3. Atau gunakan script otomatis:

start-all-servers.bat

4. Tunggu sampai muncul pesan:

CodeIgniter 4.4.8 development server started
Local: http://localhost:8081
Refresh Status Kembali ke Portal
Setelah server berjalan, refresh halaman ini untuk melanjutkan
