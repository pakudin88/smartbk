Cara Kerja Sistem

  1. User akses: /smartbk/app-guru/public
  2. .htaccess redirect: app-loader.php?app=app-guru
  3. app-loader.php baca: app-guru/.env
  4. Ambil baseURL: http://localhost:8081/
  5. cURL proxy ke: CodeIgniter server di port 8081
  6. Return content: Langsung tanpa browser redirect
  7. Result: URL tetap /smartbk/app-guru/public

Konfigurasi Aktif

app-guru/.env
app.baseURL = 'http://localhost:8081/'

Server: php spark serve --port=8081

app-ortu/.env
app.baseURL = 'http://localhost:8080/'

Server: php spark serve --port=8080
