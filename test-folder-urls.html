URL SESUAI STRUKTUR FOLDER!

URL mengikuti struktur folder asli - dinamis berdasarkan php spark serve

Struktur URL yang Benar

/smartbk/app-guru/public ← Portal Guru
/smartbk/app-ortu/public ← Portal Orang Tua
/smartbk/app-superadmin/public ← Portal Super Admin
Konsep: URL tetap sesuai struktur folder, tapi sistem dinamis membaca konfigurasi .env untuk menentukan server mana yang akan di-proxy berdasarkan php spark serve

Cara Kerja Sistem

Flow Process:
  1. User akses /smartbk/app-guru/public
  2. .htaccess route ke app-loader.php?app=app-guru
  3. app-loader.php baca app-guru/.env
  4. Ambil app.baseURL (contoh: localhost:8081)
  5. Proxy request ke server yang sedang running
  6. Return content langsung tanpa redirect
  7. URL tetap sesuai folder structure

Perbandingan URL Structure

Method URL Format Status Sesuai Request
Redirect ke Port localhost:8081 ❌ Port terlihat ❌ Tidak sesuai
Short URLs /smartbk/guru ❌ Tidak sesuai folder ❌ Tidak sesuai
Folder-Based /smartbk/app-guru/public ✅ Sesuai struktur folder ✅ Sesuai request!

Dynamic Configuration

app-guru/.env
app.baseURL = 'http://localhost:8081/'

php spark serve --port=8081

app-ortu/.env
app.baseURL = 'http://localhost:8080/'

php spark serve --port=8080

Custom Port Example
app.baseURL = 'http://localhost:9001/'

Flexible port management

Keuntungan Sistem Ini

  • Mudah diingat dan dipahami

Dynamic System
  • Port bisa diganti di .env

Cara Penggunaan

cd app-ortu
